文档名称：AI-Gal 游戏在线制作平台需求说明书

编写目的：用于前端、后端、AI 对接、测试、产品落地开发

产品定位：零代码可视化网站，用户借助各类 AI 一键生成角色、剧本、CG、BGM，在线预览后自动生成规范 Ren’Py 工程压缩包，下载即可用 RenPy 引擎打开游玩。

目标用户：Galgame 爱好者、业余创作者、不会 Python/Renpy 编程的自制玩家。

核心原则：本平台完全免费，所有功能向所有注册用户开放，无会员限制、无付费墙、无次数/容量限制。

一、用户端功能模块

模块 1：账号体系

1. 注册登录：邮箱注册、密码登录；所有功能仅面向注册用户开放。

2. 个人中心：修改头像昵称、修改密码、我的项目列表、导出记录、收藏作品。

3. 项目管理：新建空白项目、复制已有项目、重命名项目、归档 / 删除项目、项目云端自动保存。

模块 2：游戏创建分步制作（核心主流程）

步骤 1：项目基础配置

填写游戏名称、简介、作品封面、游戏分类（校园 / 奇幻 / 悬疑 / 异世界 / 日常等）、故事篇幅（短篇 3 章 / 中篇 8 章 / 长篇 15 + 章）

AI 一键生成世界观设定文案，支持手动编辑修改。

步骤 2：AI 角色生成模块

1. 新增角色：输入角色姓名、身份、性格标签，调用 AI 生成人设文案；

2. AI 绘图生成角色立绘（正装、日常、Q 版、不同表情差分），支持单张重绘、本角色全部立绘重绘、本地上传图片替换 AI 绘图；

3. 角色参数配置：出场位置（左 / 中 / 右）、角色配色备注、配音风格标注（用于后续 AI 生成语音）；

4. 角色列表：新增 / 删除 / 排序角色。

步骤 3：剧情大纲生成

用户输入核心故事梗概、结局数量（单结局 / 多分支结局）；

AI 拆分全本章节大纲、关键剧情节点、分支触发条件；

可视化拖拽调整章节顺序，新增 / 删减章节，单独重写单章大纲。

步骤 4：对话 & 分支选项编辑器（核心）

1. 基于章节大纲，AI 批量生成旁白、角色对话文本；可单段选中重新生成、调整语气（傲娇 / 冷淡 / 温柔 / 腹黑等）；

2. 分支制作：新增剧情选项，设置选项跳转目标章节，实现多分支多结局；

3. 快捷功能：插入角色出场 / 退场指令、切换背景指令、更换 BGM 标记（网页自动同步对应 Renpy 语法）。

步骤 5：场景 & 背景 CG 生成

1. 按章节场景名，AI 生成场景背景图，室内 / 室外 / 昼夜不同版本；

2. 支持本地上传自定义背景，分类收纳所有背景素材。

步骤 6：音效 / BGM 生成

1. 按场景氛围（温馨 / 紧张 / 悲伤 / 日常）AI 生成背景音乐，短音效（点击、转场、环境音）；

2. 素材库统一管理 BGM，一键绑定对应章节自动播放。

模块 3：在线实时预览功能

1. 网页模拟 Renpy 原生游玩界面：底部对话框、立绘展示、背景切换、选项弹窗；

2. 可从任意章节起点开始预览，预览时发现问题直接跳转对应编辑页面修改。

模块 4：RenPy 工程打包导出（核心）

1. 一键生成：系统自动整理全部素材 + 自动编译生成标准 .rpy 脚本文件（符合 Renpy 语法规范）；

2. 打包内容：script.rpy、image 素材文件夹、audio 音频文件夹、项目配置文件；

3. 输出：生成 ZIP 压缩包，用户下载；附带说明文档：解压后丢入 RenPy 项目目录即可启动游戏；

4. 所有用户导出无限制：无导出次数限制，无素材分辨率限制。

模块 5：素材库 & 社区（可选增值）

1. 公共素材库：创作者公开的角色、背景、BGM，一键引用到自己项目；

2. 作品广场：用户可选择将自己的 Galgame 作品公开展示，在线试玩预览。

模块 6：帮助中心

RenPy 安装教程、压缩包导入教程、网站功能使用教程、常见问题 FAQ。

二、后台管理系统

1. 用户管理：查看用户信息、封禁违规账号；

2. 作品审核：用户公开作品内容审核（风控过滤色情暴力违规内容）；

3. AI 接口配置：各类绘图 AI、大模型文案 AI、音频 AI 密钥配置、用量统计；

4. 数据统计：网站项目创建量、导出量、AI 消耗统计；

5. 内容风控：违规文案 / 图片拦截配置。

三、非功能性需求

1. 兼容：PC 端主流浏览器（Chrome/Edge 优先）；

2. 性能：单张 AI 立绘生成 ≤25s，整章对话生成 ≤15s；

3. 安全：全内容 AI 鉴黄鉴暴风控过滤，杜绝违规 Galgame 生成；

4. 存储：用户项目云端永久保存；

5. 兼容性：导出工程适配最新稳定版 Ren’Py 引擎。

3. AI 配套提示词模板 + Renpy 打包文件结构说明

（1）AI 提示词模板

① 角色人设文案提示词

按照 Galgame 人设格式创作，角色：{姓名}，标签：{性格标签}，身份：{职业 / 身份}，内容包含外貌描述、性格细节、日常习惯、人物小秘密，文风贴合日式 Galgame 文字风格，分段清晰。

② 角色立绘绘图提示词

日式二次元 Galgame 立绘，角色：{设定信息}，{表情：微笑 / 委屈 / 生气}，精细上色，全身立绘，透明背景，原画质感，CG 画风。

③ 剧情对话生成提示词

根据世界观 {世界观}、章节大纲 {章节内容}、出场角色 {角色列表}，生成 Galgame 旁白 + 角色对话，区分角色发言，文字自然贴合人物性格，不添加多余内容。

④ 场景背景提示词

Galgame 场景背景，{地点 + 环境：傍晚教室、雨夜街道、卧室}，日式插画，场景完整，无人物，高清 CG 背景。

（2）导出 RenPy 压缩包内部文件结构

```
游戏项目文件夹/
├─ game/
│  ├─ script.rpy          # 自动生成全剧情代码（核心脚本）
│  ├─ image/              # 所有角色立绘、背景CG图片
│  └─ audio/              # BGM、音效资源
├─ options.rpy            # 基础配置文件
└─ README.txt             # 导入Renpy操作说明
```

使用方式：整个文件夹导入 RenPy，编辑器打开即可一键运行游戏。
